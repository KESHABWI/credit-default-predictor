# Credit Default Predictor

A streamlined machine learning & deep learning application designed to predict the probability of credit card defaults.

## 🚀 Quick Start

Ensure you have **Docker Desktop** installed, then run:

```bash
git clone https://github.com/KESHABWI/credit-default-predictor.git
cd credit-default-predictor
docker compose up --build
```

- **Frontend**: [http://localhost:3000](http://localhost:3000)
- **FastAPI Backend**: [http://localhost:8000](http://localhost:8000) (Docs: [/docs](http://localhost:8000/docs))
- **MLflow Tracking**: [http://localhost:5001](http://localhost:5001)

---

## 🛠️ Tech Stack

- **Core ML/DL**: PyTorch, LightGBM, XGBoost, CatBoost, Scikit-learn
- **Backend**: FastAPI, PostgreSQL, MLflow
- **Frontend**: Next.js (App Router), TypeScript, Tailwind CSS
- **Orchestration**: Docker & Docker Compose
